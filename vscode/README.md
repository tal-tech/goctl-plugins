# Goctl for Visual Studio Code

## 功能列表

已实现功能

* 语法高亮
* 跳转到定义/引用
* 代码格式化
* 代码块提示

未实现功能:

* 语法错误检查
* 跨文件代码跳转
* goctl 命令行调用
